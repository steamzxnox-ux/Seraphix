# Seraphix

A keyless Roblox script with a clean built-in UI. No executor key needed, just paste and execute.

## Features

**Visuals**
- ESP — player highlights with name tags and a live health bar
- Team Check — hides teammates from ESP and aim assist

**Combat**
- Aim Assist — holds RMB to snap aim toward the nearest enemy
- Wall Check — skips targets behind walls
- Team Check — won't target teammates
- Adjustable smoothness and FOV

**Movement**
- Fly — free fly with W/A/S/D + Space/Shift, adjustable speed
- Noclip — walk through walls
- Infinite Jump
- WalkSpeed slider
- Jump Power slider

**Other**
- FPS counter (top-left, color coded green/yellow/red)
- Draggable, minimizable UI window

## Usage

Paste the script into your executor and run it. No key required.
